# 🎧 AI PDF to Audio Converter

An AI-powered web app that extracts text from PDFs, translates it into multiple languages, and converts it into speech using NLP and Text-to-Speech (TTS).

## ✨ Features

- 📄 Extract text from PDF files  
- 🌍 Translate text into multiple languages  
- 🔊 Convert text to natural-sounding audio  
- ⚡ Fast and simple web interface using Streamlit  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- NLP (Natural Language Processing)  
- Text-to-Speech (TTS)
